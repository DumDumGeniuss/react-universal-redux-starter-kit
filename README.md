# react-universal-redux-starter-kit
It's the starter kit to build up universal structure of react with redux.
We use as less code as we can to build up universal, I hope you will like it!


## DEVELOPMENT MODE

```bash
npm run start:dev
```

## PRODUCTION MODE

```bash
npm run start:prod
```

## ESLINT CHECK

```bash
npm run eslint
```
